Problem 
Develop a smart system that can analyze any GitHub repository and provide meaningful 
insights like project summary, tech stack, quality score, and improvement suggestions. 
Many developers face difficulty in: 
Understanding large codebases 
Evaluating open-source projects 
Identifying code quality and best practices 
✅
Solution  
We built RepoLense, an AI-powered GitHub repository analyzer that: 
Gives instant project understanding 
Provides health score & analytics 
Suggests improvements using AI 
Helps developers learn and evaluate code faster

Tech Stack 
Frontend: React.js, Tailwind CSS 
Backend: Node.js, Express.js
